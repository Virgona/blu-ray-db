# Blu-ray Collection Database

A personal SQL project to track my collection of films

## Features
- Normalised relational database design
- Separate tables for genres, formats, and distributors
- Foreign key constraints for data integrity
- Tracks last watched date and collection metadata

## Tech
- MySQL
- SQL

## Setup
1. Open MySQL
2. Run `sql/schema.sql`
3. Database `blu_ray_collection` will be created automatically
